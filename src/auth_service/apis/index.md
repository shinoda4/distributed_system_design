# APIs

In this section, we detail the various APIs provided by the Auth Service, including authentication mechanisms and user management functionalities.

These APIs are designed to be RESTful and can be accessed over HTTP/HTTPS. They support JSON payloads for both requests and responses.

These APIs was tested using [Postman](https://www.postman.com/).

For more details, refer to the specific API documentation linked below:

- [Authenticate](./authenticate.md)

- [User Management](./user_management.md)
